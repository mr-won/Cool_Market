# Cool_Market
국내 중고 물품 경매 앱 개발 프로젝트 - 쿨마켓
```
개발 예상 기간 : ~24년 말, kotlin Multiplatform 클라이언트, 서버, 디자인 작업
기능 : 지역별 중고물품을 올리고 경매, 종료임박 타이머
기대효과 : 구매자 입장에서 필요한 물건은 적절한 가격으로 구매해서 좋고,
판매장 입장에서는 불필요한 물건은 필요한 사람이 사가서 좋았다.

구글 플레이 스토어의 평점 및 리뷰 적극 활용, 마케팅-> 사회적으로 "선한 영향력"
거래 경제의 "선순환"

브랜딩 : "쿨 포인트" "쿨 거래 지수" "쿨 팁" "쿨 1:1 톡" 등 Cool 거래 지향
```
```
소스 코드 비공개
```

## 사용자피드백
```
경매 방법을 알려줬으면 좋겠다.
경매 물건 올려놓는 일정이 정해져 있는 것이 불편하다.-> 물품 올려놓는 경매 시간 조절(12시간, 24시간)
올라오는 물건이 한정적이다.(사용자 수가 적다)
로딩속도가 느리다.(최적화)
동네 범위를 늘린다.
상품에 구매자가 생각하고 있는 금액을 미리 저장하는 기능
상품들을 한데 모아서 보고 싶다
연령제한 상품들을 거래할 수 있는 부분

필요한 물건은 적절한 가격으로 구매해서 좋았다. 불필요한 물건은 필요한 사람이 사가서 좋았다.

판매 제품 또는 구매 제품에 따라 각 리스트 및 합산으로 나오는 기능

전번을 올리는 것이 어렵다. <- 회원가입을 간단하게 구현

매너 지수 시스템

탈퇴 시 회원정보 복구기능
```
